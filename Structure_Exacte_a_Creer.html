💻 Exercice : Je Comprends mon Ordinateur

Créez un guide visuel pour maîtriser l'architecture informatique

🎯 Le Contexte

Votre Mission

Vous êtes élève en 2e année de collège. Votre professeur informatique vous demande de créer un guide visuel complet sur l'architecture d'un ordinateur. Cet exercice va tester votre compréhension de comment fonctionne réellement un ordinateur.


Le défi : Organiser tous les composants logiciels et matériels de manière logique, trouver des images de qualité, et créer une documentation que vous pourrez expliquer à d'autres élèves. C'est comme créer un "manuel d'identité" de l'ordinateur.


📌 Pourquoi c'est important ? Comprendre les différences entre logiciels et matériel vous aidera à résoudre les problèmes informatiques et à mieux utiliser les ordinateurs dans vos études.

📋 Consignes

Ce que vous devez faire

  1. Créez des dossiers organisés qui reflètent la structure d'un ordinateur
  2. Recherchez des images claires pour chaque élément
  3. Nommez les fichiers clairement pour que tout le monde comprenne
  4. Vérifiez que tout est au bon endroit
📁 Structure Exacte à Créer
📁 SYSTEME_INFORMATIQUE/ │ ├── 📁 LOGICIELS/ │ │ │ ├── 📁 LOGICIELS_de_base/ │ │ (2 images : Windows, Linux) │ │ │ └── 📁 LOGICIELS_d'application/ │ (4 images : Word, Chrome, Teams, Excel) │ └── 📁 MATERIEL/ │ ├── 📁 Peripheriques_externes/ │ │ │ ├── 📁 Entree/ │ │ (3 images : Clavier, Souris, Webcam) │ │ │ ├── 📁 Sortie/ │ │ (3 images : Écran, Imprimante, Haut-parleurs) │ │ │ ├── 📁 Entree_Sortie/ │ │ (2 images : Routeur WiFi, Casque) │ │ │ └── 📁 Stockage/ │ (2 images : Clé USB, Disque dur externe) │ └── 📁 Composants_internes/ (3 images : Processeur, RAM, Disque SSD)

⚠️ Points clés à respecter :

  • Noms exacts : Respectez la majuscule, minuscule et les tirets bas (_)
  • Pas d'autres dossiers : Juste ce qui est demandé
